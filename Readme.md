# How to setup

## clone this repository
## open VS Code
## Press F1 key and choose option 'Dev Containers: Open Folder in Container' 
## VS Code will ask for folder path. give the path of cloned repository folder.
## let the VS Code build the image and run the container.
## do work and timely git commit and push to remote branch.

# DO NOT FORGET TO GIT PUSH TO REMOTE BEFORE LEAVING CONTAINER.
